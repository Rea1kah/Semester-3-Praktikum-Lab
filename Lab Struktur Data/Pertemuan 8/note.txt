HASH FUNCTION & HASH TABLE

=> HASH FUNCTION
- Fungsi yang mengambil input (key) dan mengembalikan output berupa indeks (hash value)
- Digunakan untuk menentukan lokasi penyimpanan data dalam hash table

=> HASH TABLE
- Struktur data yang menggunakan hash function untuk mengelola dan menyimpan data
- Terdiri dari array yang berisi bucket atau slot untuk menyimpan data
- Memungkinkan akses data yang cepat berdasarkan key

Hf = rumus => ubah kunci -> indeks array

index = key % size

=> COLLISION
- Terjadi ketika dua key yang berbeda menghasilkan hash value atau indeks yang sama 

=> METODE PENANGANAN COLLISION
1. Chaining
   - Setiap slot dalam hash table berisi linked list atau struktur data lain untuk menyimpan semua elemen yang memiliki hash value yang sama
   - Ketika terjadi collision, elemen baru ditambahkan ke linked list di slot tersebut 

2. Open Addressing
   - Ketika terjadi collision, algoritma mencari slot kosong berikutnya dalam array untuk menyimpan data baru
    - Metode ini melibatkan probing, yaitu mencari slot kosong dengan cara tertentu (linear probing, quadratic probing, double hashing)

3. Linear Probing
   - Metode open addressing di mana jika terjadi collision, algoritma mencari slot kosong berikutnya secara berurutan sampai menemukan slot yang kosong

4. Quadratic Probing
    - Metode open addressing di mana jika terjadi collision, algoritma mencari slot kosong dengan jarak yang meningkat secara kuadrat

5. Double Hashing
    - Metode open addressing di mana jika terjadi collision, algoritma menggunakan hash function kedua untuk menentukan jarak pencarian slot kosong berikutnya
